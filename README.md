A very simple retro style arcade shooter game made in HTML, CSS and JS.
This was made as a submission to a Hack Club program called ArcadAI. AI made the background stars and the enemy spawning function
and the mechanics of hitting the enemy.
I added the following:
1. Made a simple PNG in Sketchbook from scratch for the enemy starships and replaced the SVG with it.
2. Made a simple PNG from scratch of the cockpit overlay in Sketchbook and added it to simulate the effect of being inside a ship.
Making both images took around 45 minutes total.
3. Added sound effects for the laser beam firing, ship taking damage, and game over. All three sound effects I found on Pixabay.
4. Added a hearts system where every player has 5 hearts. Once all 5 are lost the game is over. For every 25 enemy ships destroyed,
   the player gains 1 heart.
5. Added an SVG crosshair and arrows pointing towards enemy ships. The arrows spawn when the ships are more than 200px away from the crosshair.
6. Added a CSS filter to the entire game to make it feel like an old CRT screen.
   It basically adds horizontal lines that continuously flicker in opacity.
7. Added a custom font called Amelia which I already had on my computer and thought it was right for the retro/sci-fi feel.
8. Added the ability to pause and resume the game.
